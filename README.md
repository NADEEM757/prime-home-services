# 🏠 Prime Home Services

Prime Home Services is a modern home service booking platform that connects customers with trusted professionals for a wide range of home maintenance and repair services. The platform is designed to provide a fast, secure, and user-friendly experience.

## ✨ Features

- 🔌 Electrician Services
- 🚰 Plumbing Services
- 🪚 Carpenter Services
- ❄️ AC Repair & Maintenance
- 🎨 Painting Services
- 🧹 Home Cleaning
- 🔍 Search Services
- 📅 Easy Booking System
- 👨‍🔧 Verified Professionals
- 💰 Affordable Pricing
- 📱 Responsive Design

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/NADEEM757/prime-home-services.git
```

2. Open the project folder.

3. If using PHP:

- Move the project to the `htdocs` folder (XAMPP).
- Import the database into phpMyAdmin.
- Start Apache and MySQL.
- Open:

```
http://localhost/prime-home-services
```
